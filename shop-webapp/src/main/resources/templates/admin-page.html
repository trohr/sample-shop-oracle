Database Console

Direct access to the database is provided through an integrated H2 console.
On login, use the following settings: 

Driver Class = oracle.jdbc.driver.OracleDriver
JDBC URL     = jdbc:oracle:thin:@localhost:1521:XE
User Name    = master
Password     = master
